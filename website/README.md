# GitHub Pages — Fendr API Anahtarı Sitesi

Bu klasörü GitHub Pages olarak yayınlayın:

1. GitHub'da `fendr-system-care-license` adında bir repo oluşturun
2. `website/` içeriğini repo köküne yükleyin
3. Settings → Pages → Source: `main` branch
4. URL: `https://bayram1233.github.io/keyken/`

Uygulamadaki `LicenseService.ActivationUrl` bu URL ile eşleşmelidir.

## Sponsor bağlantıları

`index.html` içindeki `ad-links` bölümündeki URL'leri kendi reklam/sponsor sitelerinizle değiştirin.
